### 如何搞定 Git

在你的 fork 仓库里面将 SciSharp/TensorFlow.NET 作为你的上游 upstream：

```bash
git remote add upstream git@github.com:SciSharp/TensorFlow.NET.git
```

更新一下你的 fork 仓库：

```bash
git pull upstream master
```
